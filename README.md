AI-Powered Resume & Job Tracker
A web app to manage resumes, track job applications, and parse resume content. Built with Spring Boot, a lightweight static frontend, PostgreSQL, and PDF/DOCX parsing libraries.

Features
User accounts: Signup/login with secure authentication.
Reume management: Upload PDF/DOCX, parse content for skills/experience.
Job tracking: Add jobs, track application status.
Resume parsing: Extract text from PDF/DOCX resumes for downstream analysis.

Tech Stack
Frontend: Static HTML/CSS/JS served by Spring Boot
Backend: Spring Boot, Java
Database: PostgreSQL
Parsing: PDFBox + Apache POI
Auth: Spring Security (Basic auth, JWT planned)

Usage
Sign up / log in
Upload your resume
Add jobs and track progress
Review parsed resume content
